# Graphics-Demo

## Controls:

### W/S - Go faster/stop
### K/L - toggle day/night shader
### M/N - toggle toon shader on/off
### F - first person camera
### T - third
### O - top down 
### U - unocked camera (for debugging, but use it to look around the scenery)
### Note: V/B - toggle UI, unimplemented
###
